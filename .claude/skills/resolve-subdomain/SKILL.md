# サブドメイン名解決スキル

このスキルは、ビジネスサブドメイン名を受け取り、ユビキタス言語CSVから英語名・日本語名を解決します。
他のスキルの起動時共通処理として呼び出されます。

## 実行手順

### 1. ビジネスサブドメイン名の確認

引数として渡された `$1` をビジネスサブドメイン名として扱います。`$1` が空の場合は以下のように質問してください:

```
ビジネスサブドメイン名を教えてください（例: 予約管理、会議室管理）。
```

### 2. ビジネスサブドメインの英語名の取得

`ubiquitous-finder` サブエージェントを呼び出して、ビジネスサブドメインの英語名を取得します。

```
Agent(ubiquitous-finder, "ビジネスサブドメイン: <ビジネスサブドメイン名>")
```

レスポンスのヘッダー `# ビジネスサブドメイン: {日本語名}（{英語名}）` から英語名と日本語名を抽出します。
以降の処理ではこの `<英語名>` をサブドメイン識別子（ファイルパス・モジュール名など）として使用します。

英語名が取得できない場合は、ユーザーに以下のメッセージを伝えてください:

```
ビジネスサブドメイン「<ビジネスサブドメイン名>」がユビキタス言語CSVに見つかりません。
正しいサブドメイン名を指定して再度お試しください。
```

このスキルが完了した後、呼び出し元のスキルは取得した `<英語名>` と `<日本語名>` を使って処理を継続します。
