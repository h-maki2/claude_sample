# ユビキタス言語 検索スキル

このスキルは、プロジェクトのユビキタス言語を管理するCSVファイルから、ユーザーが指定したビジネスサブドメインに属する用語を抽出します。

## スキルの目的
- 指定されたビジネスサブドメインに属する用語を全て取得する
- ビジネスサブドメインとユビキタス言語（日本語名 or 英語名）から、ユビキタス言語の意味や使い方を理解するための情報を提供する

## モード1: ユビキタス言語一覧取得

### 目的
ユーザーが指定したビジネスサブドメインに属する全てのユビキタス言語を一覧表示します。

### 実行手順

1. 引数 `$1` をビジネスサブドメインとして使用します。`$1` が空の場合はユーザーに質問してください。
2. `ubiquitous_language.csv` のパスを特定してください。不明な場合は `Glob` ツールで探すか、ユーザーにパスを尋ねてください。
3. 同梱されている `search_language.py` スクリプトを使用して、以下のコマンドで検索を実行します。

```bash
python3 search_language.py --csv-path "2番で特定したCSVのパス" --domain "$1"
```

### 使い方・実行イメージ

Claude Codeを起動し、以下のようにビジネスサブドメインを渡して話しかけるだけで、Claudeが自動的にスキルを読み込み、スクリプトを実行して結果を返してくれます。

ビジネスサブドメインは**日本語名・英語名どちらでも検索できます**（大文字小文字を区別しません）。

**入力例：**
> 「/ubiquitous-language 予約管理」
> 「/ubiquitous-language ReservationManagement」

## モード2: ビジネスサブドメインとユビキタス言語から、ユビキタス言語の意味や使い方を理解するための情報提供

### 目的
ユーザーが指定したビジネスサブドメインとユビキタス言語（日本語名 or 英語名）から、ユビキタス言語の意味や使い方を理解するための情報を提供します。

### 実行手順

1. 引数 `$1` をビジネスサブドメイン、`$2` をユビキタス言語名として使用します。いずれかが空の場合はユーザーに質問してください。
2. `ubiquitous_language.csv` のパスを特定してください。不明な場合は `Glob` ツールで探すか、ユーザーにパスを尋ねてください。
3. 同梱されている `search_language.py` スクリプトを使用して、以下のコマンドで検索を実行します。

```bash
python3 search_language.py --csv-path "2番で特定したCSVのパス" --domain "$1" --name "$2"
```

### 使い方・実行イメージ

Claude Codeを起動し、以下のようにビジネスサブドメインとユビキタス言語名を渡して話しかけるだけで、Claudeが自動的にスキルを読み込み、スクリプトを実行して結果を返してくれます。

ビジネスサブドメインは**日本語名・英語名どちらでも指定できます**（大文字小文字を区別しません）。

**入力例：**
> 「/ubiquitous-language 予約管理 予約」
> 「/ubiquitous-language ReservationManagement Reservation」
